# 双链笔记

双链笔记功能是该项目的核心，使用`tiptap`包实现类似Notion的编辑体验，同时与当前项目中的论文相关功能结合。

笔记支持Notion中的常见功能，如插入公式（支持单$的行内公式与$$的行间公式）、代码、表格，同时还可以通过命令链接文献、任务与其他笔记，笔记还可以导出为markdown。

笔记管理页面设置在 `/note`（笔记管理）下，按更新时间顺序展示所有笔记（分页表格），同时配备根据笔记名的搜索框。用户可在该页面创建、删除已有笔记，并查看每条笔记的基本信息（如标题、创建时间、更新时间）。编辑按钮会导向专门的编辑器界面，该界面可以是另一种布局，即不需要Navbar，编辑框尽可能大以保证良好的编辑体验。

每篇笔记的编辑页面置于 `/note/{id}` 下，页面右上角为返回首页按钮，下方区域左侧大部分为编辑器区域，会随着内容增加扩大，右侧边栏（固定，不随编辑器容器滚动）显示该笔记的链接，包括文献、任务与其他笔记，文献链接表现为卡片形式，必有arxiv链接按钮，若有精读报告则包含精读报告按钮，点击后弹窗显示精读报告正文，包含两个视图，第一个视图为渲染后的markdown，第二个视图为markdown源码（方便用户复制）。弹窗大小固定，可滚动。任务则包含一个查看详情按钮，点击后弹窗查看该任务的具体描述。

创建笔记时默认标题为“未命名笔记”，若已存在标题则后面自动加序号。编辑器抬头不需要加工具栏，快捷插入格式均采用/command的形式。在编辑时按ctrl+s保存，页面上方显示当前保存状态，保存时须有反馈。支持ctrl+z回退，ctrl+shift+z取消回退的功能。
