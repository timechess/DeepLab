# 论文初筛

对收集得到的每日论文进行初筛，分为两阶段筛选：

1. 根据论文upvotes数目，仅保留前15篇论文。
2. 将前15篇论文的标题、摘要、ai关键词发送给初筛Agent进行筛选。

初筛Agent的prompt包含如下部分：

1. 角色设定：包含任务描述与指引。
2. 论文信息：15篇论文的基本信息。
3. 初筛规则：数据库中记录的初筛规则，可由用户自行添加或由记忆Agent自动添加。
4. 格式约定：输出的格式。

其中初筛规则在数据库中的格式如下：

| 属性      | 描述     |
| --------- | -------- |
| id        | 规则id   |
| rule      | 规则描述 |
| createdBy | 用户或AI |
| createdAt | 创建时间 |
